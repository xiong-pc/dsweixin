# Claude Code 命令参考

## 会话管理

| 命令 | 说明 |
|------|------|
| `/clear` | 清除对话历史，释放上下文。别名：`/reset`、`/new` |
| `/exit` | 退出 CLI。别名：`/quit` |
| `/help` | 显示帮助和可用命令列表 |
| `/compact [说明]` | 压缩对话内容，可指定保留重点 |
| `/resume [会话]` | 恢复之前的会话（按 ID 或名称），或打开会话选择器。别名：`/continue` |
| `/rewind` | 将对话和代码回退到之前的某个检查点。别名：`/checkpoint` |
| `/rename [名称]` | 重命名当前会话，不填则自动生成名称 |
| `/branch [名称]` | 在当前位置创建对话分支。别名：`/fork` |

## 模型与性能配置

| 命令 | 说明 |
|------|------|
| `/model [模型名]` | 切换 AI 模型 |
| `/effort [low\|medium\|high\|max\|auto]` | 设置模型推理强度（max 仅 Opus 4.6 可用） |
| `/fast [on\|off]` | 切换快速模式开/关 |

## 代码与项目管理

| 命令 | 说明 |
|------|------|
| `/init` | 初始化项目，生成 `CLAUDE.md` 文件 |
| `/diff` | 打开交互式 diff 查看器，查看未提交的变更 |
| `/simplify` | 审查已改动文件的代码质量并自动修复 |
| `/security-review` | 对当前分支上的待提交变更进行安全漏洞分析 |
| `/plan [描述]` | 进入计划模式，先规划再执行 |

## 权限管理

| 命令 | 说明 |
|------|------|
| `/permissions` | 管理工具权限的允许/询问/拒绝规则。别名：`/allowed-tools` |
| `/add-dir <路径>` | 为本次会话临时添加可访问的工作目录 |

## 自动化与工作流

| 命令 | 说明 |
|------|------|
| `/loop [间隔] [提示词]` | 循环定期执行某个提示词。别名：`/proactive` |
| `/schedule [描述]` | 创建、列出或运行定时任务 |
| `/batch <指令>` | 用 5~30 个并行 Agent 对代码库执行大规模批量变更 |

## 信息与监控

| 命令 | 说明 |
|------|------|
| `/context` | 用彩色图表可视化当前上下文用量 |
| `/cost` | 显示本次会话的 Token 用量统计 |
| `/stats` | 展示每日用量、会话历史、连续使用天数等统计 |
| `/insights` | 生成对你 Claude Code 使用记录的分析报告 |
| `/export [文件名]` | 将当前对话导出为纯文本文件 |
| `/copy [N]` | 复制最近一条回复到剪贴板 |
| `/tasks` | 列出和管理后台任务。别名：`/bashes` |
| `/hooks` | 查看工具事件的 hook 配置 |

## 设置与配置

| 命令 | 说明 |
|------|------|
| `/config` | 打开设置界面（主题/模型/输出风格）。别名：`/settings` |
| `/theme` | 切换颜色主题（亮色/暗色/色盲友好/ANSI 主题等） |
| `/keybindings` | 打开或创建快捷键配置文件 |
| `/terminal-setup` | 配置终端快捷键（如 Shift+Enter） |
| `/memory` | 编辑 CLAUDE.md 记忆文件，管理自动记忆条目 |

## IDE 与环境集成

| 命令 | 说明 |
|------|------|
| `/ide` | 管理 IDE 集成状态 |
| `/desktop` | 在 Claude Code 桌面应用中继续当前会话。别名：`/app` |
| `/mcp` | 管理 MCP 服务器连接与 OAuth 认证 |

## 账号与认证

| 命令 | 说明 |
|------|------|
| `/login` | 登录 Anthropic 账号 |
| `/logout` | 登出 Anthropic 账号 |
| `/usage` | 查看套餐用量限制和速率限制状态 |
| `/upgrade` | 打开升级页面切换到更高套餐 |

## 工具与诊断

| 命令 | 说明 |
|------|------|
| `/doctor` | 诊断并验证 Claude Code 安装和设置是否正常 |
| `/feedback` | 提交反馈或 Bug 报告。别名：`/bug` |
| `/release-notes` | 查看更新日志 |
| `/skills` | 列出当前可用的 Skill 命令 |
| `/voice` | 切换语音输入（按住说话）模式 |

---

> 提示：在终端输入 `/` 后可自动补全可用命令列表。标注 **[Skill]** 的命令是内置技能提示词，由 Claude 自动调用。
